== DESCRIPTION:

Easily execute ruby code in parallel

== SYNOPSIS:

By default there is maximum of 10 simultaneous threads

  urls.in_threads.each do |url|
    url.save_to_disk
  end

  urls.in_threads.map do |url|
    url.fetch
  end

  numbers.in_threads(2).map do |number|
